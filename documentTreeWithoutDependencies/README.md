# Make Sure

### Npm Install

```

```
### Install bootstrap

```
npm i bootstrap --save
```
### Install ngx-markdown

```
npm install ngx-markdown --save
```
### Install fontawesome

```
npm i @fortawesome/fontawesome-free --save
```

### Install lodash

```
npm i lodash --save
```
