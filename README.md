# Pre-requirement

## Install Nodejs

## Install Angular cli

## proxy issue

### 1. proxy add
```
npm config set proxy http://<>:<>@<>:8080
npm config set https-proxy http://<>:<>@<>:8080
npm config set registry=https://registry.npmjs.org
```

### 2. proxy remove

```
npm config rm proxy
npm config rm https-proxy
```
